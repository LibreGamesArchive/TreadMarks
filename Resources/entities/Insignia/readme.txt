Team Insignia.

Perturb = ferocity of perturbation (amount each wave moves flag).

PerturbScale = size of individual "waves" in perturbation.

PerturbSpeed = waves per second for flag to move through.

Rotation = pitch, heading, roll base rotation for flag.

Team = 1, 2, 3, 4, 5, etc. for TEAM banners.  Set to 0 (or leave unspecified) for personal insignia.
